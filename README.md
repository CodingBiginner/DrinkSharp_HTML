## 필요한 데이터 목록
### 1. 회원쪽 필요한 데이터
> member table (회원 테이블)
> (회원코드(PK), 이메일, 이름, 비밀번호, 전화번호, 우편번호, 기본주소, 상세주소, 적립금, 생년월일, 가입날짜, 이메일 및 SMS 마케팅 수신 여부)
